# Pattern Maker 🔓

Un visualiseyr de "Pattern Lock" (schéma de déverrouillage) super simple et léger.

J'ai créé ce petit outil à la base pour mon frère qui en avait besoin pour générer rapidement des schémas de verrouillage de multiples appareils avec un rendu propre, et j'ai décidé d'en faire un site prêt à l'emploi.

## ✨ Fonctionnalités

* **Tracé fluide :** Reliez les points à la souris ou au doigt (compatible mobile/tablette).
* **Rendu technique :** Affiche automatiquement l'ordre des points (1, 2, 3...) et des flèches directionnelles sur les segments.
* **Look Futuriste :** Interface sombre/claire avec la police "Orbitron" pour un côté dashboard/tech.
* **Export Haute Définition :** Téléchargement du tracé en `.png` avec un fond transparent. L'image est générée en haute résolution (900x900px) pour être parfaitement nette, même sur mobile.
* **Fichiers personnalisés :** Possibilité de taper un nom avant l'export pour que le fichier téléchargé s'appelle `NomDeLaPersonne.png`.

## 🚀 Comment l'utiliser ?

C'est un système globalement assez simple pour visualiser un code par schéma en 3x3 :

1. Entrez un nom (optionnel)
2. Tracez votre schéma
3. Cliquez sur **Télécharger PNG**.

## 🛠️ Technologies utilisées

* **HTML5 Canvas** (pour le dessin libre et l'export HD)
* **CSS3** (design intégré)
* **JavaScript** (logique de tracé sans aucune librairie externe)
